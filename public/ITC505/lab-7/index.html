Space Adventure Mad Lib

Challenges & Resolutions

During the implementation of this Mad Lib project, I encountered the following challenges and resolved them as follows:

  • Handling Form Data on the Server Side: I initially struggled to retrieve and process the form data properly. I fixed this by ensuring that I used express.urlencoded({ extended: true }) to parse the form data correctly.
  • Route Conflict with Static Files: There was a conflict between serving static files and defining custom routes in my server.js. I fixed this by ensuring that static file routes were defined before custom routes.
  • Preventing Empty Form Submissions: Initially, I didn't handle empty form submissions well, resulting in incomplete Mad Libs. I added form validation in the backend to ensure all fields were filled before generating the output.
  • Displaying Mad Lib Output Correctly: The output wasn't rendering correctly due to improper string concatenation. I resolved this by using string interpolation and improving the HTML structure for better readability.
